# Horizons Arcana

Horizons Arcana is a 3-day long Hack Club hackathon for 100+ teens happening in Singapore this summer (July 31st - August 2nd). I created this static website for the event using HTML, CSS and a little in-line JavaScript.
<br><br>
All art was made by @Cin on the Hack Club Slack.
